Основна інформація про лабораторну:

Розраховані параметри С3 та С17, у зв'язку с тим, що я 1 у списку групи, усі параметри дорівнюють 1.
За варіантом було визначено тип текстових змінних, а саме String;

Основне завдання:
"Вивести всі речення заданого тексту в порядку зростання кількості слів у них."

Для зчитування даних с консолі було використано клас Scanner, що надає необхідні можливості.

Для того, щоб розбити текст на речення я використав регулярний вираз "(?<=[.!?])\s*".
Цей регулярний вираз знаходить поділ після знаків закінчення речень, ігноруючи будь-які прогалини,
і використовується для коректного розбиття тексту на речення.

Для того, щоб виділити слова у реченнях я використав регулярний вираз "[\s,;:—\-]+".
Цей регулярний вираз відповідає одному або більше пробілових символів, ком, крапок з комою, двокрапками,
тире та їх комбінаціям.

Змінна text зберігає у собі інформацію про введену інформацію з консолі.
Масив sentences містить у собі реченні, отримані поділом тексту.

Код оформлено відповідно Google Java Style Guide.

Для використання коду можна використати git clone.
